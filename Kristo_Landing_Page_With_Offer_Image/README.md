# Kristo Landing Page

صفحة هبوط عربية سريعة ومتجاوبة لربط إعلانات كريستو بمنصات الطلب.

## الملفات
- `index.html`
- `styles.css`
- `script.js`

## النشر على GitHub Pages
1. أنشئ Repository جديد على GitHub.
2. ارفع الملفات الثلاثة إلى الصفحة الرئيسية للمستودع.
3. افتح:
   Settings → Pages
4. اختر:
   Deploy from a branch
5. اختر:
   Branch: main / Folder: root
6. احفظ وانتظر ظهور رابط الموقع.

## مهم قبل استخدام الرابط في TikTok Ads
- افتح الصفحة من الجوال واختبر الزرين.
- استخدم رابط الصفحة الجديدة كـ Website URL في الإعلان.
- لا تستخدم رابط Keeta أو HungerStation مباشرة كوجهة الإعلان.
- لا تضع رقم واتساب أو رقم هاتف داخل الإعلان إذا كان يسبب رفضًا آليًا.
- لا يوجد ضمان قبول تلقائي؛ المراجعة النهائية بيد TikTok.

## تعديل النصوص
افتح `index.html` وعدّل:
- عنوان العرض
- السعر
- وصف المطعم
- روابط Keeta وHungerStation

## إضافة صورة طبق حقيقية
استبدل الجزء داخل `.dish-card` بصورة فعلية، أو أضف ملفًا باسم `hero.jpg` ثم استخدم:
```html
<img src="hero.jpg" alt="عرض الفروج المشوي">
```
